Widget props with loading state

Generic interface for widgets that load and display data asynchronously.

## Example

```typescript
interface MyData {
  value: number;
}

const props: WidgetPropsWithLoading<MyData> = {
  data: { value: 42 },
  loading: false,
  error: undefined
};
```
